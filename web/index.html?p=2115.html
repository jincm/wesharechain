    WeShare链动态 || WeShare链有望让月饼企业迎来信赖“升级”

    发布时间：2017/09/23 新闻报道 热门推荐 浏览次数：691

    近些年来，国家对食品安全的把控越来越严，各级政府更是竭尽全力为老百姓的饮食安全保驾护航，并取得了显著的成绩。所以，国家领导人多次就加强和创新社会治理作出重要指示，强调“提高预测预警防各类风险能力”,这样对于食品安全领域的社会治理同样具有重要的指导意义。各地执法部门相继加强对食品行业上游企业的突击检查，从食品原材料开始把关，从源头上遏制可能出现的安全风险等等。

    同时，各大食品企业更是从自身做起，以强烈的社会责任心，不遗余力地提升产品质量。这一点，从中秋各大品牌月饼企业的宣传上就可见一斑。而最近，一种新的溯源体系—WeShare链的兴起，有望将这种大好局面再推进一步。为月饼企业带来信赖“升级”。

    据悉，WeShare链与市面上其他溯源体系的区别在于，整个溯源过程的可信度更高，操作更方便。WeShare链一位负责人告诉小编，WeShare链产品主要包含溯源、存证、查真和限购(分布式商城)等四大体系。其中溯源产品包括智能物联网组件、外部物联网适配网关、业务管理子系统等，实现了对食品从生产到消费全过程数据的采集、实时上报进行分布式存储，并通过大数据智能分析实现了数据的监测、追踪防窜货、风险预警和食品事件应急响应处理等；存证产品实现了数据保真性存储、食品事件责任追踪与认定等；查真产品实现了对食品溯源信息的查看，了解食品的真实性；限购商城产品为消费者提供了放心食品的购买渠道，为生产者提供了优质食品销售渠道，实现了双向互通互信和互赢。

    据小编进一步了解：WeShare链溯源系统整体主要分为三层：Sensing Layer, Data Storage Layer, Application Layer。Sensing Layer主要是自主设计的IOT传感器模块，传感器实时采集数据并上传写入区块链系统中，Application Layer主要是用于实现客户化应用（溯源、存证、查真、限购电商等）。

    感知层IOT模块主要包括温度传感器、湿度传感器、加速度传感器、压力传感器、GPS定位、GPRS模块。传感器感知周围环境，可以采集温度湿度以及位置变化情况，同时加速度传感器用于感知商品运输状态，压力传感器感知气压变化，一旦产品密封包装被打开，就会导致气压变化，压力传感器就能捕获异常并上传数据。这也就有效的解决了产品被调包，或不良经销商串货等问题。

    中秋临近，如果月饼企业能够抢先使用WeShare链体系，势必会在这个旺季节点上脱颖而出，通过强有力的产品保证，让消费者吃着放心，让不良的竞争对手无可乘之机。
